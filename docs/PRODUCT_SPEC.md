# StayFlow Product Specification v0.6

## 1. Цель продукта

StayFlow заменяет хаос WhatsApp, Excel, звонков и ручных списков единой операционной системой для краткосрочной аренды.

Платформа соединяет:

- владельцев недвижимости;
- управляющие компании;
- уборщиков;
- мастеров;
- гостей;
- администраторов платформы.

## 2. Главная ценность

### Для Owner

- контроль объектов;
- автоматические задачи;
- меньше ручной координации;
- фотоотчеты;
- финансы;
- прозрачность команды.

### Для Worker

- стабильные заказы;
- понятное расписание;
- адреса и инструкции;
- фотоотчеты;
- доход и история выплат.

### Для Guest

- быстрый доступ к инструкциям;
- Wi-Fi, код двери, правила;
- чат;
- заказ услуг;
- сообщение о проблеме.

## 3. MVP

В MVP входят:

1. Регистрация и вход.
2. Объекты недвижимости.
3. Бронирования.
4. Автоматическая уборка после выезда.
5. Задачи для Worker.
6. Фотоотчеты.
7. Guest Link.
8. Базовый чат.
9. Уведомления.
10. Финансовая сводка.

## 4. Что не входит в MVP

- сложная интеграция Airbnb/Booking;
- Stripe payments;
- AI concierge;
- динамическое ценообразование;
- полноценный marketplace;
- бухгалтерия;
- электронная подпись.

Это идет во v2.
