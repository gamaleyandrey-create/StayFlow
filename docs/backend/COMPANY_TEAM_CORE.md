# Company + Team Core v1.5

## Added

- Company model
- User.companyId
- Companies module
- Team module
- Invite manager/worker
- List team members

## Main flow

1. Owner registers.
2. Owner creates company.
3. Owner invites manager.
4. Owner invites worker.
5. Future properties are scoped to company.
