# StayFlow Role Permissions

## OWNER

Can:

- create company;
- manage subscription;
- create/edit/delete properties;
- create/edit/delete bookings;
- create/edit/delete tasks;
- assign workers;
- view finance;
- manage team;
- view reports;
- access all chats.

## MANAGER

Can:

- create/edit properties;
- create/edit bookings;
- create/edit tasks;
- assign workers;
- chat with guests/workers;
- view limited finance.

Cannot:

- delete company;
- manage subscription;
- remove owner;
- access full billing settings.

## WORKER

Can:

- view assigned tasks;
- view available marketplace tasks;
- accept task;
- start task;
- complete task;
- upload photos;
- report problem;
- chat with manager;
- view own earnings.

Cannot:

- see company finance;
- see unrelated properties;
- see all guests;
- edit booking data.

## GUEST

Can:

- open own guest link;
- view check-in info;
- view apartment guide;
- chat with owner/manager;
- request service;
- report problem.

Cannot:

- see other bookings;
- see internal tasks;
- see worker info unless exposed.

## ADMIN

Can:

- manage all users;
- manage companies;
- review platform data;
- manage subscriptions;
- inspect abuse reports;
- access support tools.
