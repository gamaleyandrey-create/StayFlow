# v2.1 Guest Web App MVP

## MVP approach

Guest app should be web-first.

Guest opens:

```text
/guest/:guestLink
```

No login required.

## Screens

1. Welcome
2. Check-in Info
3. Wi-Fi
4. Door Code
5. Apartment Guide
6. Chat
7. Services
8. Report Problem
9. Checkout Info

## Security

Guest link expires after checkout + 24h.
