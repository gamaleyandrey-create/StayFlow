# Guest App MVP Screens

## 1. Guest Link Loading

Validates guest link.

If valid:

- open Welcome.

If expired:

- show expired message.

## 2. Welcome

Shows:

- guest name
- property title
- stay dates
- property photo
- Start button

## 3. Check-in Info

Shows:

- address
- check-in time
- door code
- key location
- parking
- support contact

## 4. Wi-Fi

Shows:

- Wi-Fi name
- Wi-Fi password
- copy button

## 5. Apartment Guide

Sections:

- air conditioner
- washing machine
- trash
- TV
- kitchen
- house rules

## 6. Chat

Guest can send:

- text
- photo

## 7. Report Problem

Quick buttons:

- door problem
- no hot water
- air conditioner
- internet
- other

## 8. Services

MVP request buttons:

- extra cleaning
- towels
- late checkout
- transfer
- other
