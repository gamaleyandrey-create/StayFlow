# Definition of Done

A feature is done when:

- API endpoint works.
- UI screen exists.
- Basic validation exists.
- Errors are handled.
- Data is saved or loaded correctly.
- Role access is respected.
- Documentation is updated.
- At least one manual test is passed.

## MVP is done when:

- Owner can create property.
- Owner can create booking.
- System auto-creates cleaning task.
- Owner can assign worker.
- Worker can complete task with photos.
- Guest can open guest link.
- Guest can report problem.
- Owner sees problem/task.
