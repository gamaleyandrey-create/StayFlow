# Flow: Worker Cleaning

1. Booking checkout date arrives.
2. System creates cleaning task.
3. Owner assigns worker or marketplace offers task.
4. Worker accepts.
5. Worker starts work.
6. Worker completes checklist.
7. Worker uploads required photos.
8. Worker submits report.
9. System changes task to COMPLETED.
10. Property status changes to READY.
11. Owner receives notification.
